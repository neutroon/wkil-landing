# Wkil Landing

Standalone Next.js landing page for **وكيل / wkil**.

## Structure

- `src/app/` — App Router entry, metadata, and global styles
- `src/components/LandingPage.tsx` — bilingual landing page UI
- `src/content.ts` — Arabic Egyptian and English copy
- `public/assets/` — Wkil logo assets served by Next
- `index.html`, `styles.css`, `app.js`, `assets/` — legacy static source kept for reference

## Commands

```bash
npm install
npm run dev
npm run typecheck
npm run build
```
